# [spring 1편] mapstruct 활용



## mapstruct 무엇인가?

- mapstruct 은 객체 맵핑을 도와주는 라이브러리입니다.
- 예를 들면, dto <-> dto 간 코드 맵핑을 도와주는 것입니다.
