# QPS (Query per second)

- 초당 검색 횟수입니다.



### Example

- 1일 = 24시간 * 60분 * 60 초 = 86,400 초 -> 약 10⁵
- 사용자가 천만명 가정, 하루 10번 조회한다고 가정합니다.
- 10,000,000 * 10 / 100,000 = 1000 QPS
